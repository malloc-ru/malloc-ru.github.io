Hello World на Си

Из этой статьи вы узнаете как написать программу, которая выводит сообщение "Hello, world".
В качестве языка программирования я выбрал - С (си).
Тема далеко не нова, но все же у многих возникают трудности при написании этой программы.

Все действия протестированы и проверены на дистрибутиве Ubuntu.
В статье есть видео, в которых всё показано в живую.
Приятного чтения и просмотра!

Подготовка

    Шаг 1
  • Открываем терминал:
    "Ctrl+Alt+T" или "ПКМ, открыть Терминал" (по англ. - Open in Terminal).

    • Шаг 2
  • Создадим отдельный каталог (папку):
    mkdir prog
    Теперь зайдем в него:
    cd prog Мы создали пустой каталог и зашли в него для того, чтобы ничего не мешало.

    "mkdir" - создать пустой каталог
    "cd" - используется для перемещения по каталогам


    • Шаг 3
  • В этой папке создадим файл "hello.c":
    touch hello.c
    Убидимся, что файл создался:
    ls

    "touch" - создать файл
    "ls" - просмотр текущего каталога


    • Шаг 4
    Теперь нужно открыть файл в любой текстовом редакторе.
    К примеру открою наш файл "hello.c" в текстовом редакторе - gedit.
  • Нажимаем "Ctrl+Shift+T", для того, чтобы открыть новую вкаладку в терминале.
    Теперь вводим следующую команду:
    gedit hello.c & exit
    Откроется "текстовой редактор - gedit" и закроется одна вкладка в терминале.

Пишем код

    Я напишу три примера вывода сообщения в терминал - Hello, world.
    После каждого примера нужно запустить компиляцию.
    Пример 1
    Пишите код строчка за строчкой:
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    Далее пишем функцию - main:
    int main()
    Открывающая скобка:
    {
    Теперь пишем приветствие:
    printf("Hello, world");
    Чтобы наша строка не слилась со строкой в терминале, добавим переход на красную строку:
    printf("\n");
    Далее (по правилам) вернем целое число:
    return 0;
    Закрывающая скобка:
    }

    • Пример 2
    Пишите код строчка за строчкой:
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    Далее пишем функцию - main:
    void main()
    Открывающая скобка:
    {
    Теперь пишем приветствие:
    printf("Hello, world\n");
    Закрывающая скобка:
    }

    • Пример 3
    Пишите код строчка за строчкой:
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    Пишем макрос - hello:
    #define hello printf("Hello, world\n");
    Далее пишем функцию - main:
    void main() { hello }

Компиляция и запуск

    Чтобы совершить сборку программы нужен - "компилятор gcc".
    Установим его:
    sudo apt install gcc
    Если он у вас уже есть, то хорошо, идем дальше.

    Шаг 1
  • Компиляция:
    gcc hello.c
    Посмотрим, что появилось в нашем каталоге:
    ls
    В каталоге дожны лежать "a.out" и "hello.c".
    • Шаг 2
  • Теперь посмотрим на результат:
    ./a.out
    Если все сделано правильно, то увидим сообщение:
    Hello, world

Заключение

  • В данной статье была расмотрена программа "Hello, world" на языке программирования С (си).
    В статье приведены три примера, в которых можно увидеть, что одно и тоже сообщение выводится в терминал разными способами.
    В итоге мы собрали нашу программу и запустили её на исполнение.
