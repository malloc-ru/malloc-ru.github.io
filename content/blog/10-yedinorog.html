Единорог

В этой статье я расскажу как объединять разные программы в единое целое.
Для это я использую последние три статьи, которые похоже тематике
Объединю их код в одну программу.

  • "Отжимания в отказ. Программа"
  • "Три золотых упражнения. Программа"
  • "Распределение максимального количествава повторений на подходы"

В качестве языка программирования для статьи я выбрал - С. Программа также написана и на С++.
Все действия протестированы и проверены на дистрибутиве Ubuntu.
В статье есть видео, в которых всё показано в живую.

Приятного чтения и просмотра!

Что будет делать программа

  • Запустив программу мы увидим заголовок, меню.
    Пункты меню выбраем при помощи клавиш от 1 до 3.
    После нажатий "Enter", ниже появиться заголовок и меню выбранной программы.

    Вот и всё, что она будет делать. Приступим к сборке!

    Код программ из предыдущих статей:
    C - "первая", "вторая", "третья"

Подготовка

  • Шаг 1
  • Открываем терминал:
    "Ctrl+Alt+T" или "ПКМ, открыть Терминал" (по англ. - Open in Terminal).
  • Шаг 2
  • Создадим отдельный каталог (папку):
    mkdir c-prog
    Теперь зайдем в него:
    cd c-prog
    Мы создали пустой каталог и зашли в него для того, чтобы ничего не мешало.

    "mkdir" - создать пустой каталог
    "cd" - используется для перемещения по каталогам

  • Шаг 3
  • В этой папке создадим несколько файлов - "main.c", "file-1.c", "file-2.c", "file-3.c", "header.h", "mymakfile":
    touch main.c file-1.c file-2.c file-3.c header.h mymakfile
    Убидимся, что файлы создались:
    ls

    "touch" - создать файл(ы)
    "ls" - просмотр текущего каталога

  • Шаг 4
  • Теперь нужно открыть файл в любой текстовом редакторе.
    К примеру открою наш файл "main.c" в текстовом редакторе - gedit.
  • Нажимаем "Ctrl+Shift+T", для того, чтобы открыть новую вкаладку в терминале.
    Теперь вводим следующую команду:
    gedit main.c & exit
    Откроется "текстовой редактор - gedit" и закроется одна вкладка в терминале.

Пишем в файле 'main.c'

  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
  • Функция main
  • Отступаем вниз пару строк и пишем функцию main:
    int main()
    Открывающая скобка:
    {
    Нам понадобиться переменная для выбора пункта меню:
    int num;

    Пишем заголовок и меню:
    printf("\tЕдинорог\n\n");
    printf("1. Отжимания в отказ\n");
    printf("2. Три золотых упражнения\n");
    printf("3. Распределение максимальноого кол-ва повторений на подходы\n\n");

    Пишем приглашение на ввод:
    printf("Выбор: ");
    scanf("%d", &num);

    Заполняем switch():
    switch(num)
    Открывающая скобка "switch":
    {
    У нас три программы. Их функции на выводе назвались "main()".
    Дадим им другие имена - "firstProg", "secondProg", "thirdProg":
    case 1: firstProg(); break;
    case 2: secondProg(); break;
    case 3: thirdProg(); break;
    Пишем исключения:
    default:
    printf("\nчто то пошло не так =(\n")
    Закрывающая скобка "switch":
    }
    Возвращаем целое число:
    return 0;
    Закрывающая скобка:
    }

Откроем файл 'file-1.c'

  • Пишем программу из статьи "Отжимания в отказ. Программа" и кореектируем код под наши нужды.
    Я напишу здесь всесь код, а изменненным строкам уделю особое внимание!
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    Далее пишем функцию - firstProg. Она не чего не передает, поэтому делаем её void.
    Да эта та самая функция, которая будет передаваться в main():
    void firstProg()
    Открывающая скобка:
    {
    Нам понадобиться четыре переменные.
    "Отжимания", "Подтягивания", "Приседания", "Скручивания на пресс":
    int otzhimaniya, podtyagivaniya, prised, skruchivaniya;

    Пишем заголовок:
    printf("\tОтжимания в отказ, определение нагрузки.\n");

    Приглашаем пользоваеля ввести кол-во отжиманий:
    printf("Кол-во отжиманий: ");
    scanf("%d", &otzhimaniya);

    Пишем формулы из условия.
    Для подтягиваний:
    podtyagivaniya = otzhimaniya/4;
    Для приседаний:
    prised = otzhimaniya*2;
    Для скручиваний на пресс:
    skruchivaniya = prised;

    Выводим результат на экран:
    printf("Подтягивания = %d\n", podtyagivaniya);
    printf("Приседания = %d\n", prised);
    printf("Скручивания = %d\n", skruchivaniya);

    Закрывающая скобка:
    }

Откроем файл 'file-2.c'

  • Пишем программу из статьи "Три золотых упражнения. Программа" и кореектируем код под наши нужды.
    Я напишу здесь всесь код, а изменненным строкам уделю особое внимание!
  • Функция отжимания
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    Далее пишем функцию отжимания:
    int otzhimaniya()
    Открывающая скобка:
    {
    Нам понадобиться четыре переменные.
    "Результат", "Рост", "Вес", "Возраст":
    int result, rost, ves, vozrast;

    Записываем нуль в переменную "result":
    result = 0;

    Далее пишем приглашение на ввод:
    printf("Ваш рост (см): ");
    scanf("%d", &rost);
    printf("Ваш вес (кг): ");
    scanf("%d", &ves);
    printf("Ваш возраст: ");
    scanf("%d", &vozrast);

    Пишем формулу из условия для отжимания и сразу возвращаем результат переменной:
    return result=(rost+ves)/((vozrast+10)/10);
    Закрывающая скобка функции otzhimaniya():
    }
  • Функция подтягивания
  • Отступаем вниз пару строк и пишем функцию подтягивания:
    int podtyagivaniya()
    Открывающая скобка:
    {
    Дальше идет тот же код, что и в функции otzhimaniya(), но возвращаем теперь формулу для подтягиваний:
    return result=(rost+ves)/vozrast;
    Закрывающая скобка функции podtyagivaniya():
    }
  • Функция приседания
  • Отступаем вниз пару строк и пишем функцию приседания:
    int prisedaniya()
    Открывающая скобка:
    {
    Нам понадобиться две переменные.
    "Результат", "Возраст":
    int result, vozrast;
    Записываем нуль в переменную "result":
    result = 0;

    Далее пишем приглашение на ввод:
    printf("Ваш возраст: ");
    scanf("%d", &vozrast);

    Пишем формулу из условия для приседания и сразу возвращаем результат переменной:
    return result=120-vozrast;
    Закрывающая скобка функции prisedaniya():
    }
  • Функция secondProg
  • Далее пишем функцию - secondProg. Она не чего не передает, поэтому делаем её void.
    Эта функция, которая будет передаваться в main():
    void secondProg()
    Открывающая скобка:
    {
    Пишем заголовок и меню:
    printf("\tТри золотых упражнения.\n");
    printf("1. Отжимания\n");
    printf("2. Подтягивания\n");
    printf("3. Приседания\n\n");

    Нам нужна переменная в которую будем записывать номер пункта меню:
    int num;

    Пишем приглошения на выбор пункта меню:
    printf("Выбор: ");
    scanf("%d", &num);

    Теперь опишем "switch", чтобы он правильно реагировал на выбор пункта меню:
    switch(num)
    Открывающая скобка "switch":
    {
    Вывод для первого пункт меню:
    case 1: printf("Кол-во отжиманий = %d\n", otzhimaniya()); break;
    Второго пункта меню:
    case 2: printf("Кол-во подтягиваний = %d\n", podtyagivaniya()); break;
    Третьего пункта меню:
    case 3: printf("Кол-во приседаний = %d\n", prisedaniya()); break;
    Пишем исключения для ввода:
    default:
    printf("\nчто то пошло не так =(\n");
    Закрывающая скобка "switch":
    }
    Закрывающая скобка:
    }

Откроем файл 'file-3.c'

  • Пишем программу из статьи "Распределение максимального количества повторений на подходы. Программа" и кореектируем код под наши нужды.
    Я напишу здесь всесь код, а изменненным строкам уделю особое внимание!
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
  • Функция - два подхода
  • Далее пишем функцию "два подхода":
    int twoSet(int max)
    Пишем формулу из условия для "двух подходов" и сразу возвращаем результат переменной:
    { return max*=0.80; }
  • Функция - три подхода
  • Далее пишем функцию "три подхода":
    int threeSet(int max)
    Пишем формулу из условия для "трёх подходов" и сразу возвращаем результат переменной:
    { return max*=0.65; }
  • Функция - четыре подхода
  • Далее пишем функцию "четыре подхода":
    int fourSet(int max)
    Пишем формулу из условия для "четырех подходов" и сразу возвращаем результат переменной:
    { return max*=0.50; }
  • Функция - пять подходов
  • Далее пишем функцию "пять подходов":
    int fiveSet(int max)
    Пишем формулу из условия для "пяти подходов" и сразу возвращаем результат переменной:
    { return max*=0.35; }
  • Функция - шесть подходов
  • Далее пишем функцию "шесть подходов":
    int sixSet(int max)
    Пишем формулу из условия для "шести подходов" и сразу возвращаем результат переменной:
    { return max*=0.20; }
  • Функция thirdProg
  • Далее пишем функцию - thirdProg. Она не чего не передает, поэтому делаем её void.
    Эта функция, которая будет передаваться в main():
    void thirdProg()
    Открывающая скобка:
    {
    Для передачи в функции, которые мы написали выше, нам понадобиться переменная:
    int max;

    Пишем заголовок и меню:
    printf("\tРаспределение максимальноого кол-ва повторений на подходы\n\n");
    Пишем приглошение на ввод:
    printf("Ваше кол-во повторений: ");
    scanf("%d", &max);

    Вывод для "два подхода":
    printf("\t2 по %d.\n", twoSet(max));
    Вывод для "три подхода":
    printf("\t3 по %d.\n", threeSet(max));
    Вывод для "четыре подхода":
    printf("\t4 по %d.\n", fourSet(max));
    Вывод для "пять подходов":
    printf("\t5 по %d.\n", fiveSet(max));
    Вывод для "шесть подходов":
    printf("\t6 по %d.\n", sixSet(max));
    Закрывающая скобка:
    }

Откроем файл 'header.h'

  • Способ заполнения заголовочного файла можете использовать свой. А я использую вот такой метод. Он универсальный:
    #ifndef __HEADER_H__
    #define __HEADER_H__

    Пишем наши функции:
    void firstProg();
    void secondProg();
    void thirdProg();

    Завершаем наше условие. Комментарии служат для отметки, какое именно условие завершино:
    #endif // __HEADER_H_

Отрываем файл 'main.c'

  • После строчки - #include <stdio.h> пишем:
    #include "header.h"
    Мы включили файл 'header.h' и теперь функции описанные в нем можно использовать по назначению.

Работаем с файлом 'mymakefile'

  • Открываем файл и пишем
  • all:
    Теперь нажимаем Enter и нажимаем Tab.
  • Пишем все в одну строчку.
    Указываем компилятор 'gcc' и ключи по вкусу:
    gcc -Wall -g
    Далее записываем файлы с расширением - .с:
    main.c file-1.c file-2.c file-3.c
    Даем имя нашей программе:
    -o unicorn
  • Закрываем файл и переходим в терминал
  • Если вы закрыли терминал, то открываем новый и заходим в нашу папку:
    cd c-prog

    Работаем в терминале.
    Нам нужно дать права на исполнение файлу 'mymakefile'. Иначе наш скрипт не будет работать:
    chmod +x mymakefile

Сборка и запуск

  • Чтобы совершить сборку программы воспользуемся - "утилита make".
  • Шаг 1
  • Собираем программу при помощи make:
    make -f mymakefile
    Убидимся, что файл "unicorn" появился в нашем каталоге:
    ls
  • Шаг 2
  • Запустим программу:
    ./unicorn
    Если все сделано правильно, то увидим заголовок меню и приглашение на ввод.
