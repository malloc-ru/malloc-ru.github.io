Три золотых упражнения. Программа

Из этой статьи вы узнаете как создать программу подсчитывающая количество приседаний, подтягиваний, отжиманий исходя из вашего возраста, веса, роста.

В качестве языков программирования я выбрал - С и С++.
Все действия протестированы и проверены на дистрибутиве Ubuntu.
В статье есть видео, в которых всё показано в живую.

Приятного чтения и просмотра!

Формулы

  • 1. Отжимания= ("рост" + "вес") / (("возраст" + 10) / 10)
    2. Подтягивания= ("рост" + "вес") / "возраст"
    3. Приседания= 120 - "возраст"

Подготовка для Си

  • Шаг 1
  • Открываем терминал:
    "Ctrl+Alt+T" или "ПКМ, открыть Терминал" (по англ. - Open in Terminal).
  • Шаг 2
  • Создадим отдельный каталог (папку):
    mkdir c-prog
    Теперь зайдем в него:
    cd c-prog
    Мы создали пустой каталог и зашли в него для того, чтобы ничего не мешало.

    "mkdir" - создать пустой каталог
    "cd" - используется для перемещения по каталогам

  • Шаг 3
  • В этой папке создадим файл "main.c":
    touch main.c
    Убидимся, что файл создался:
    ls

    "touch" - создать файл
    "ls" - просмотр текущего каталога

  • Шаг 4
  • Теперь нужно открыть файл в любой текстовом редакторе.
    К примеру открою наш файл "main.c" в текстовом редакторе - gedit.
  • Нажимаем "Ctrl+Shift+T", для того, чтобы открыть новую вкаладку в терминале.
    Теперь вводим следующую команду:
    gedit main.c & exit
    Откроется "текстовой редактор - gedit" и закроется одна вкладка в терминале.

Пишем код на Си

  • Функция отжимания
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    Далее пишем функцию отжимания:
    int otzhimaniya()
    Открывающая скобка:
    {
    Нам понадобиться четыре переменные.
    "Результат", "Рост", "Вес", "Возраст":
    int result, rost, ves, vozrast;

    Записываем нуль в переменную "result":
    result = 0;

    Далее пишем приглашение на ввод:
    printf("Ваш рост (см): ");
    scanf("%d", &rost);
    printf("Ваш вес (кг): ");
    scanf("%d", &ves);
    printf("Ваш возраст: ");
    scanf("%d", &vozrast);

    Пишем формулу из условия для отжимания и сразу возвращаем результат переменной:
    return result=(rost+ves)/((vozrast+10)/10);
    Закрывающая скобка функции otzhimaniya():
    }
  • Функция подтягивания
  • Отступаем вниз пару строк и пишем функцию подтягивания:
    int podtyagivaniya()
    Открывающая скобка:
    {
    Дальше идет тот же код, что и в функции otzhimaniya(), но возвращаем теперь формулу для подтягиваний:
    return result=(rost+ves)/vozrast;
    Закрывающая скобка функции podtyagivaniya():
    }
  • Функция приседания
  • Отступаем вниз пару строк и пишем функцию приседания:
    int prisedaniya()
    Открывающая скобка:
    {
    Нам понадобиться две переменные.
    "Результат", "Возраст":
    int result, vozrast;
    Записываем нуль в переменную "result":
    result = 0;

    Далее пишем приглашение на ввод:
    printf("Ваш возраст: ");
    scanf("%d", &vozrast);

    Пишем формулу из условия для приседания и сразу возвращаем результат переменной:
    return result=120-vozrast;
    Закрывающая скобка функции prisedaniya():
    }
  • Функция main
  • Отступаем вниз пару строк и пишем функцию main:
    int main()
    Открывающая скобка:
    {
    Пишем заголовок и меню:
    printf("\tТри золотых упражнения.\n");
    printf("1. Отжимания\n");
    printf("2. Подтягивания\n");
    printf("3. Приседания\n\n");

    Нам нужна переменная в которую будем записывать номер пункта меню:
    int num;

    Пишем приглошения на выбор пункта меню:
    printf("Выбор: ");
    scanf("%d", &num);

    Теперь опишем "switch", чтобы он правильно реагировал на выбор пункта меню:
    switch(num)
    Открывающая скобка "switch":
    {
    Вывод для первого пункт меню:
    case 1: printf("Кол-во отжиманий = %d\n", otzhimaniya()); break;
    Второго пункта меню:
    case 2: printf("Кол-во подтягиваний = %d\n", podtyagivaniya()); break;
    Третьего пункта меню:
    case 3: printf("Кол-во приседаний = %d\n", prisedaniya()); break;
    Пишем исключения для ввода:
    default:
    printf("\nчто то пошло не так =(\n");
    Закрывающая скобка "switch":
    }

    Возвращаем целое число:
    return 0;
    Закрывающая скобка:
    }

Компиляция и запуск

  • Чтобы совершить сборку программы нужен - "компилятор gcc".
    Установим его:
    sudo apt install gcc
    Если он у вас уже есть, то хорошо, идем дальше.
  • Шаг 1
  • Компиляция:
    gcc main.c
    Посмотрим, что появилось в нашем каталоге:
    ls
    В каталоге дожны лежать "a.out" и "main.c".
  • Шаг 2
  • Теперь посмотрим на результат:
    ./a.out
    Если все сделано правильно, то увидим заголовок меню и приглашение на ввод.
    Выберем каждый пункт поочереди и посмотрим результат.

    Выбор: 1
    Ваш рост (см): 180
    Ваш вес (кг): 92
    Ваш возраст: 34
    Кол-во отжиманий = 68

    Выбор: 2
    Ваш рост (см): 162
    Ваш вес (кг): 58
    Ваш возраст: 21
    Кол-во подтягиваний = 10

    Выбор: 3
    Ваш возраст: 45
    Кол-во подтягиваний = 75

Пишем код на С++

  • Функция отжимания
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <iostream>
    Подключаем пространство имен - "std":
    using namespace std;
    Далее пишем функцию отжимания с тремя переменными - рост, вес, возраст:
    int otzhimaniya (int rost, int ves, int vozrast)
    Открывающая скобка:
    {
    Пишем формулу из условия для отжимания и возвращаем результат:
    return (rost+ves)/((vozrast+10)/10);
    Закрывающая скобка:
    }
  • Функция подтягивания
  • Отступаем вниз пару строк и пишем функцию подтягивания с тремя переменными - рост, вес, возраст:
    int podtyagivaniya (int rost, int ves, int vozrast)
    Открывающая скобка:
    {
    Пишем формулу из условия для подтягивания и возвращаем результат:
    return (rost+ves)/vozrast;
    Закрывающая скобка:
    }
  • Функция приседания
  • Отступаем вниз пару строк и пишем функцию приседания с одной переменной - возраст:
    int prisedaniya(int vozrast)
    Открывающая скобка:
    {
    Пишем формулу из условия для приседания и возвращаем результат:
    return 120-vozrast;
    Закрывающая скобка:
    }
  • Функция main
  • Отступаем вниз пару строк и пишем функцию main:
    int main()
    Открывающая скобка:
    {
    Нам понадобиться три переменные - рост, вес, возраст:
    int rost, ves, vozrast;

    Пишем заголовок:
    cout << "\tТри золотых упражнения.\n";

    Приглашенние на ввод "Рост":
    cout << "Рост: ";
    cin >> rost;
    Приглашенние на ввод "Вес":
    cout << "Вес: ";
    cin >> ves;
    Приглашенние на ввод "Возраст":
    cout << "Возраст: ";
    cin >> vozrast;

    Отступ между вводом и выводом в терминале:
    cout << endl;
    Вывод на экран "Отжимания":
    cout << "Отжимания = " << otzhimaniya (rost, ves, vozrast) << endl;
    "Подтягивания":
    cout << "Подтягивания = " << podtyagivaniya (rost, ves, vozrast) << endl;
    "Приседания":
    cout << "Приседания = " << prisedaniya (vozrast) << endl;

    Возвращаем целое число:
    return 0;
    Закрывающая скобка:
    }
