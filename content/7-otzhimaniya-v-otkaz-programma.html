Отжимания в отказ. Программа

Из этой статьи вы узнаете как создать программу подсчитывающая количество приседаний, подтягиваний, скручиваний.
Основной переменной служит количество отжиманий.

Программа создана для ознакомительных целей, хотя можно использовать и в других накинув GUI.

В качестве языка программирования я выбрал - С и С++.
Все действия протестированы и проверены на дистрибутиве Ubuntu.
В статье есть видео, в которых всё показано в живую.

Приятного чтения и просмотра!

Смотрим условие

  • "Класические отжимания от пола"
    - cделайте одно повторение до отказа.
    - запишите разультат.

    Далее следуйте следующей формуле:
    - четыре отжимания это одно подтягивание.
    - приседаний в 2 раза больше чем отжиманий.
    - скручивания на пресс равны количеству приседаний.

Подготовка для Си

  • Шаг 1
  • Открываем терминал:
    "Ctrl+Alt+T" или "ПКМ, открыть Терминал" (по англ. - Open in Terminal).
  • Шаг 2
  • Создадим отдельный каталог (папку):
    mkdir c-prog
    Теперь зайдем в него:
    cd c-prog
    Мы создали пустой каталог и зашли в него для того, чтобы ничего не мешало.

    "mkdir" - создать пустой каталог
    "cd" - используется для перемещения по каталогам

  • Шаг 3
  • В этой папке создадим файл "main.c":
    touch main.c
    Убидимся, что файл создался:
    ls

    "touch" - создать файл
    "ls" - просмотр текущего каталога

  • Шаг 4
  • Теперь нужно открыть файл в любой текстовом редакторе.
    К примеру открою наш файл "main.c" в текстовом редакторе - gedit.
  • Нажимаем "Ctrl+Shift+T", для того, чтобы открыть новую вкаладку в терминале.
    Теперь вводим следующую команду:
    gedit main.c & exit
    Откроется "текстовой редактор - gedit" и закроется одна вкладка в терминале.

Пишем код

  • Пишите код строчка за строчкой.
  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    Далее пишем функцию - main:
    int main()
    Открывающая скобка:
    {
    Нам понадобиться четыре переменные.
    "Отжимания", "Подтягивания", "Приседания", "Скручивания на пресс":
    int otzhimaniya, podtyagivaniya, prised, skruchivaniya;

    Пишем заголовок:
    printf("\tОтжимания в отказ, определение нагрузки.\n");

    Приглашаем пользоваеля ввести кол-во отжиманий:
    printf("Кол-во отжиманий: ");
    scanf("%d", &otzhimaniya);

    Пишем формулы из условия.
    Для подтягиваний:
    podtyagivaniya = otzhimaniya/4;
    Для приседаний:
    prised = otzhimaniya*2;
    Для скручиваний на пресс:
    skruchivaniya = prised;

    Выводим результат на экран:
    printf("Подтягивания = %d\n", podtyagivaniya);
    printf("Приседания = %d\n", prised);
    printf("Скручивания = %d\n", skruchivaniya);

    Далее (по правилам) вернем целое число:
    return 0;
    Закрывающая скобка:
    }

Компиляция и запуск

  • Чтобы совершить сборку программы нужен - "компилятор gcc".
    Установим его:
    sudo apt install gcc
    Если он у вас уже есть, то хорошо, идем дальше.
  • Шаг 1
  • Компиляция:
    gcc main.c
    Посмотрим, что появилось в нашем каталоге:
    ls
    В каталоге дожны лежать "a.out" и "main.c".
  • Шаг 2
  • Теперь посмотрим на результат:
    ./a.out
    Если все сделано правильно, то увидим заголовок. А на другой строче приглашение на ввод.

    Кол-во отжиманий: 20
    Подтягивания = 5
    Приседания = 40
    Скручивания = 40
