Распределение максимального количества повторений на подходы

Из этой статьи вы узнаете как создать программу, которая распределит максимальное количество повторений на подходы.

Отжались 54 раза. Распределили это число на 3 подхода по 65% от максимума.
И теперь мы можем тренироваться так: 3 серии по 35!

В качестве языков программирования я выбрал - С и С++.
Все действия протестированы и проверены на дистрибутиве Ubuntu.
В статье есть видео, в которых всё показано в живую.

Приятного чтения и просмотра!

Формулы

  • 2 подхода это 80% от максимума = "макс. повтор." * 0.80
    3 подхода это 65% от максимума = "макс. повтор." * 0.65
    4 подхода это 50% от максимума = "макс. повтор." * 0.50
    5 подхода это 35% от максимума = "макс. повтор." * 0.35
    6 подхода это 20% от максимума = "макс. повтор." * 0.20

    Получаем количество повтрений за один подход.

Подготовка для Си

    Шаг 1
  • Открываем терминал:
    "Ctrl+Alt+T" или "ПКМ, открыть Терминал" (по англ. - Open in Terminal).

    • Шаг 2
  • Создадим отдельный каталог (папку):
    mkdir c-prog
    Теперь зайдем в него:
    cd c-prog
    Мы создали пустой каталог и зашли в него для того, чтобы ничего не мешало.

    "mkdir" - создать пустой каталог
    "cd" - используется для перемещения по каталогам


    • Шаг 3
  • В этой папке создадим файл "main.c":
    touch main.c
    Убидимся, что файл создался:
    ls

    "touch" - создать файл
    "ls" - просмотр текущего каталога


    • Шаг 4
    Теперь нужно открыть файл в любой текстовом редакторе.
    К примеру открою наш файл "main.c" в текстовом редакторе - gedit.
  • Нажимаем "Ctrl+Shift+T", для того, чтобы открыть новую вкаладку в терминале.
    Теперь вводим следующую команду:
    gedit main.c & exit
    Откроется "текстовой редактор - gedit" и закроется одна вкладка в терминале.

Пишем код на Си

  • Подключаем стандартную библиотеку ввода-вывода:
    #include <stdio.h>
    • Функция - два подхода
  • Далее пишем функцию "два подхода":
    int twoSet(int max)
    Пишем формулу из условия для "двух подходов" и сразу возвращаем результат переменной:
    { return max*=0.80; }

    • Функция - три подхода
  • Далее пишем функцию "три подхода":
    int threeSet(int max)
    Пишем формулу из условия для "трёх подходов" и сразу возвращаем результат переменной:
    { return max*=0.65; }

    • Функция - четыре подхода
  • Далее пишем функцию "четыре подхода":
    int fourSet(int max)
    Пишем формулу из условия для "четырех подходов" и сразу возвращаем результат переменной:
    { return max*=0.50; }

    • Функция - пять подходов
  • Далее пишем функцию "пять подходов":
    int fiveSet(int max)
    Пишем формулу из условия для "пяти подходов" и сразу возвращаем результат переменной:
    { return max*=0.35; }

    • Функция - шесть подходов
  • Далее пишем функцию "шесть подходов":
    int sixSet(int max)
    Пишем формулу из условия для "шести подходов" и сразу возвращаем результат переменной:
    { return max*=0.20; }

    • Функция main
  • Отступаем вниз пару строк и пишем функцию main:
    int main()
    Открывающая скобка:
    {
    Для передачи в функции, которые мы написали выше, нам понадобиться переменная:
    int max;

    Пишем заголовок и меню:
    printf("\tРаспределение максимальноого кол-ва повторений на подходы\n\n");
    Пишем приглошение на ввод:
    printf("Ваше кол-во повторений: ");
    scanf("%d", &max);

    Вывод для "два подхода":
    printf("\t2 по %d.\n", twoSet(max));
    Вывод для "три подхода":
    printf("\t3 по %d.\n", threeSet(max));
    Вывод для "четыре подхода":
    printf("\t4 по %d.\n", fourSet(max));
    Вывод для "пять подходов":
    printf("\t5 по %d.\n", fiveSet(max));
    Вывод для "шесть подходов":
    printf("\t6 по %d.\n", sixSet(max));
    Возвращаем целое число:
    return 0;
    Закрывающая скобка:
    }

Компиляция и запуск

    Чтобы совершить сборку программы нужен - "компилятор gcc".
    Установим его:
    sudo apt install gcc
    Если он у вас уже есть, то хорошо, идем дальше.

    Шаг 1
  • Компиляция:
    gcc main.c
    Посмотрим, что появилось в нашем каталоге:
    ls
    В каталоге дожны лежать "a.out" и "main.c".
    • Шаг 2
  • Теперь посмотрим на результат:
    ./a.out
    Если все сделано правильно, то увидим заголовок меню и приглашение на ввод.
    Посмотрим результат.

    Ваше кол-во повторений: 22
    2 по 17.
    3 по 14.
    4 по 11.
    5 по 7.
    6 по 4.

Заключение

  • В статье расмотрена программа распределяющая количество повторений на подходы. Чем больше подходов, тем меньше повторений в сете.
    Вы сами можете отрегулировать процентовку в подходах, тем самым сделав более правильный вывод в консоль.

    Автор использовал для создания программы языки программирования "С", "С++".
    Программа на Си отличается от программы на С++, но формулы остались неизменны.
